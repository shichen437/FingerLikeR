## ✨ 项目简介

FignerLike R 是一个基于 [Tauri](https://tauri.app/) 和 [Vue3](https://vuejs.org/) 构建的鼠标连点器。

## 🚀 功能特性

- ❤️ **点击模式**: 支持仿生点赞/普通点击/自定义模式。
- 🏝️ **闲动模式**: 防网页休眠检测。

## 📺 支持平台

- darwin-aarch64
